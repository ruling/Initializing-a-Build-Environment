Mokee OpenSource 开发环境配置脚本，使用谷歌推荐的open java 7

使用方法：

打开打开终端，输入以下命令给予权限：

sudo chmod 775 IABE.sh

再输入以下命令。

./IABE.sh

然后根据提示进行配置。
